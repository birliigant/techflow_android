# Apache Answer 接口文档

接口文档建议按下面顺序查看：

- 总体接口与权限分层：本文件
- 每个接口的请求参数、请求体模型、成功响应模型： [接口文档-详细版.md](接口文档-详细版.md)
- 原始 Swagger 产物： [swagger.yaml](./swagger.yaml) 和 [swagger.json](./swagger.json)

## 说明

- `api-reference-detailed.md` 基于当前仓库代码和 `docs/swagger.json` 整理
- 已额外补充 Swagger 未覆盖但代码中真实存在的接口
- 已修正已知的 Swagger 注解偏差，例如 `GET /answer/api/v1/user/notification/config`

## 推荐查看方式

如果你要对接接口：

1. 先看 [接口文档-详细版.md](接口文档-详细版.md) 定位接口
2. 再看对应的请求体/响应体模型定义
3. 如果和 Swagger 有出入，以代码实际注册路由为准
